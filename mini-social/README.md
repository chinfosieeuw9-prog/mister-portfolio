# Mini Social Platform — Startertemplate

Dit is een professionele basis voor een mini-social media platform:

## Features (eerste opzet)
- Tijdlijn met berichten (posts)
- Nieuwe berichten plaatsen (in-memory demo)
- Centrale header/footer en versiebeheer
- Responsive, modern design
- Uitbreidbaar met profielen, likes, reacties, volgers, enz.

## Structuur
- `index.html` — Home/tijdlijn
- `css/style.css` — Styling
- `js/version.js` — Versiebeheer
- `js/components.js` — Header/footer componenten
- `js/main.js` — Tijdlijnlogica
- `profiles/` — (toekomstig) gebruikersprofielen
- `posts/` — (toekomstig) opslag van posts
- `assets/` — Afbeeldingen/icons

## Volgende stappen
- Profielpagina’s toevoegen
- Likes/reacties/volgers uitbreiden
- (Optioneel) Backend voor opslag
