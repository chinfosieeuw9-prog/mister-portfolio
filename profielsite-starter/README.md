# Profielsite Starter Template

Deze map bevat een minimale, professionele blauwdruk voor een profielsite. Kopieer of pas aan voor nieuwe projecten.

## Structuur

- `index.html` — Hoofdpagina, laadt header/footer en versie dynamisch
- `css/style.css` — Centrale styling
- `js/version.js` — Centraal versienummer
- `js/components.js` — Herbruikbare UI-componenten
- `js/main.js` — Algemene JS-logica
- `assets/` — Afbeeldingen, icons, etc.
- `logs/` — (optioneel) logging/changelog
- `api/` — (optioneel) backend scripts

## Checklist professionaliteit

- [x] Centraal versiebeheer (`js/version.js`)
- [x] Uniforme header/footer componenten
- [x] Responsive layout (pas `style.css` aan)
- [ ] Logging/changelog (optioneel)
- [ ] Uitbreidbaar met cards, modals, etc.
- [ ] Duidelijke projectdocumentatie

## Gebruik
1. Pas `index.html` aan voor je eigen content.
2. Zet je naam in de footer (`components.js`).
3. Werk het versienummer bij in `version.js`.
4. Voeg extra pagina's toe naar wens.
