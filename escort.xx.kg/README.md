# AI Dating Site — escort.xx.kg

Dit is een eerste opzet voor een exclusieve, AI-gedreven datingsite.

## Features (eerste opzet)
- Moderne, uitnodigende layout
- Profiel aanmaken (placeholder)
- AI-matchmaking (voorbereid, nog niet geïmplementeerd)
- Centrale plek voor versiebeheer
- Uitbreidbaar met chat, profielbeheer, invite-only, enz.

## Structuur
- `index.html` — Homepagina, AI-match overzicht
- `css/style.css` — Styling
- `js/version.js` — Versiebeheer
- `js/components.js` — Header/footer componenten
- `js/main.js` — Basislogica, profiel aanmaken placeholder
- `profiles/` — (toekomstig) gebruikersprofielen
- `api/` — (toekomstig) backend/API
- `assets/` — Afbeeldingen/icons

## Volgende stappen
- Profiel aanmaken en opslaan
- AI-matching logica toevoegen
- (Optioneel) Chatfunctie, invite-only, privacy features
